# Avatar

Avatar can be used to represent users, collaborators or objects.
You can render `image`, `icon` or `text` inside Avatars.

## Usage

The Avatar component can be imported using

```jsx
import { Avatar } from '@tail-kit/tail-kit'
```

### Avatars in Navbar

<Playground
  editable
  scope={{
    users,
    AiOutlineEdit,
    AiOutlineFontSize,
    FiFigma,
    IoMdHand,
    FaRegComment,
  }}
>
  {function Navbar() {
    return (
      <div className="flex items-center space-x-4 justify-between w-full px-3 py-2 bg-gray-800 rounded">
        <FiFigma className="w-6 h-6 text-white" />
        <div className="h-6 border-r border-white" />
        <AiOutlineEdit className="w-4 h-4 text-white" />
        <AiOutlineFontSize className="w-4 h-4 text-white" />
        <IoMdHand className="w-4 h-4 text-white" />
        <FaRegComment className="w-4 h-4 text-white" />
        <div className="flex-1" />
        <Avatar.Group
          maxCount={4}
          excessAvatarStyle={{ color: '#f56a00', backgroundColor: '#fde3cf' }}
          maxPopoverPlacement="bottom"
        >
          {users.map((user, index) => (
            <Tooltip title={user.name} placement="bottom" key={index}>
              <Avatar src={user.profileImage} />
            </Tooltip>
          ))}
        </Avatar.Group>
        <Button buttonType="primary">Share</Button>
      </div>
    )
  }}
</Playground>
