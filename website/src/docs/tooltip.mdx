# Tooltip

Component to show **tooltip**.

- The tip is shown on mouse enter, and is hidden on mouse leave. The Tooltip doesn't support complex text or operations.
- To provide an explanation of a button/text/operation. It's often used instead of the html title attribute.

## Usage

The Tooltip component can be imported using

```jsx
import { Tooltip } from '@tail-kit/tail-kit'
```

### Default Tooltip

<Playground scope={{ HiOutlinePlus }} editable>
  <Tooltip title="Create a new blog">
    <Button icon={<HiOutlinePlus />} />
  </Tooltip>
</Playground>

### Tooltip with icon

<Playground scope={{ HiExclamationCircle, HiOutlineTrash }}>
